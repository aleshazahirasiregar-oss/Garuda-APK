# Garuda-APK Documentation

## 📖 Overview
Garuda-APK adalah game absurd "Tahan Kencing 💧🚻" yang dibuat dengan Python.  
Tujuan game: menjaga bladder meter agar tidak penuh sambil menghadapi event random.

## 🗂️ Struktur Repo
- `src/` → kode utama (main.py, bladder.py, teleport.py, events.py, sprites.py)
- `assets/` → sprite dan gambar
- `docs/` → dokumentasi tambahan
- `README.md` → dokumentasi utama di root
- `LICENSE` → lisensi project

## 🎮 Gameplay Flow
1. Bladder naik setiap detik.
2. Event random muncul tiap bladder % 30.
3. Di 50% → karakter mulai gelisah.
4. Di 80% → teleport ke toilet.
5. Di 100% → game over (kencing celana).
